# TikTok Türkiye Game

## Setup Instructions

### Prerequisites
- Make sure you have the following installed on your system:
  - Node.js
  - Git

### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/alihansuperonline-png/tiktok-t-rkiye-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd tiktok-t-rkiye-
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Game Rules

### Basic Rules
- Each player takes turns to participate in challenges.
- Players must adhere to the game's time limits.

### Game Objectives
- Complete the challenges to earn points.
- The player with the most points at the end of the game wins.

## Usage Guide

### How to Start the Game
1. Run the following command to start the game:
   ```bash
   npm start
   ```

### Controls and Features
- Follow the on-screen instructions to navigate through the game.
- Use your keyboard to make selections during challenges.
